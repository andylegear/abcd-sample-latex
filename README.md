# ABCD Sample LaTeX Document

Test repository for use with the [ABCD LaTeX editor](https://andylegear.github.io/abcdLatex/).

## Usage

1. Go to https://andylegear.github.io/abcdLatex/
2. Paste your GitHub fine-grained PAT (with Contents read/write on this repo)
3. Enter `andylegear/abcd-sample-latex` as the repository
4. Open `main.tex`
